# Phishing-Email-Detection-Model
This project is a machine learning-based phishing email detection system developed using Python and Scikit-learn. The model analyzes email content and classifies messages as either "Phishing" or "Safe".
## Features

- Email classification using Machine Learning
- TF-IDF text feature extraction
- Phishing and legitimate email detection
= Accuracy score calculation
- Confusion matrix generation
##Technologies Used
- Python
- Scikit-learn
- Pandas
- TF-IDF Vectorizer
- Multinomial Naive Bayes
## Project Structure

├── phishing_detector.py

├── emails.csv

├── report.txt

└── README.md
